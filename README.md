This project analyzes coffee shop sales data using Microsoft Excel.
An interactive dashboard was created to track sales performance, customer behavior, and product trends.

Data Cleaning

Removed duplicates
Handled missing values
Formatted date & time
Extracted Month Name, Day Name, and Hour
Structured data for Pivot analysis

KPIs Created

Total Sales
Total Footfall
Average Bill per Person
Average Orders per Person

Dashboard Visualizations

Quantity Orders based on Hours
Category % Distribution based on Sales
Size % Distribution based on Orders
Footfall & Sales by Store Location
Top 5 Products based on Sales
Orders on Weekdays

Interactive Filters

Month Name Slicer
Day Name Slicer

Tools Used
Microsoft Excel, Pivot Tables, Pivot Charts, Slicers, Calculated Measures
